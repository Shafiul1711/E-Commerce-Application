Switching Themes in PuzzleMania

Administrators can change the color of the website by following the steps below:

  1. Log in as an admin.
  2. Go to the Admin Panel.
  3. Click on Theme Settings.
  4. Select a theme from the dropdown (e.g., theme_fall.css, theme_dark.css, or theme_default.css).
  5. Click Apply Theme.

The selected theme the selected them will then be applied on refresh

All available theme files are located in:

  • /assets/css/theme_default.css
  • /assets/css/theme_dark.css
  • /assets/css/theme_fall.css

To add more themes, place a new theme_*.css file in that folder and it will automatically appear in the dropdown.
